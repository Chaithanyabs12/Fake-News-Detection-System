
# Fake News Detection System

A Machine Learning based web application developed using Flask and Scikit-learn to detect whether a news article is real or fake based on textual content analysis and natural language processing techniques.

## Technologies Used
- Python
- Flask
- HTML/CSS
- Scikit-learn
- Pandas
- NumPy
- NLP (Natural Language Processing)

## Features
- Fake News Classification
- Real-time News Prediction
- Text Preprocessing & Vectorization
- Machine Learning Model Integration
- Simple Responsive User Interface

## Project Modules
- Data Cleaning
- Text Preprocessing
- Feature Extraction
- Model Training
- News Prediction System

## Future Enhancements
- Live News API Integration
- Deep Learning-based Detection
- Multi-language Support
- News Source Credibility Analysis
